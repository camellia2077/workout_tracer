## [v0.3.1] - 2026-03-28

### 新增功能 (Added)
* 新增 `apps/workout_io/src/` 与 `workout_io` 静态库，承载文本解析、校验、持久化与报表导出相关实现。

### 技术改进/重构 (Changed/Refactor)
* 重构 `apps/workout_core/src` 与 `apps/workout_io/src` 的职责边界，将 `DatabaseHandler` 与 `ActionHandler` 改为通过 `IWorkoutRepository`、`IReportExporter` 编排 IO 能力。
* 重构 Python 工具链与测试入口，从 `scripts/` 迁移到 `tools/`，并将 `test/suites/workout_calculator/tests.toml` 收敛为 TOML 驱动的命令套件。
* 重构 CLI 为 `主命令 + 子命令 + 分层 --help` 结构，命令树切换到 `data`、`query`、`report`。

### 修复 (Fixed)
* 修复 `test/run.py`、Shell 脚本与格式化流程的入口路径解析，统一改用 `tools/run.py`。
* 修复 CLI 在未知命令、未知子命令与缺参场景下的提示策略，改为输出就近 `--help` 指引。

### 弃用/删除 (Deprecated/Removed)
* 删除空的 `scripts/` 目录与旧 `apps/workout_windows_cli/src/cli/command_line_parser.*` 平铺解析路径。
* 删除 `docs/history/v0.3.md` 单文件记录方式，改为 `docs/history/0.3/0.3.0.md` 与后续 patch 文件。
