## [v0.1.0] - 2026-02-25

### Added
* 新增 **Android MVP**：在 `apps/workout_android/app/src/main/` 复用 `workout_core`，打通插入与查询主流程。
* 新增 **Query 结果渲染**：支持 `Markdown` 展示，点击卡片后在页面内展开内容。
* 新增 **导航结构**：底部提供 `Insert`、`Query`、`Config` 三个 `Tab`。
* 新增 **版本信息展示**：在 `Config` 展示 `CPP Core Version` 与 `Android Presentation Version`。
* 新增 **主题设置持久化**：在 `Config` 支持 `Light`、`Dark`、`Follow System` 与彩虹顺序主色选择，并通过 `Jetpack DataStore (Preferences)` 持久化到 `workout_user_prefs.preferences_pb`。

### Changed/Refactor
* 重构 **Config 布局**：将 `Appearance` 与 `About` 分区展示，避免信息混排。
* 优化 **主题配色注入**：在 `MainActivity` 基于 `ThemeMode` 与 `AccentColor` 动态生成 `MaterialTheme` 的 `colorScheme`。
